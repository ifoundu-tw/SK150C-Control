SK150C Modbus 控制台 - v25 原型

執行檔：
SK150C_Control_v25.exe

v25 更新：
- 左欄新增「語言」切換，可選中文 / English。
- 預設語言為中文。
- 切換語言後會即時更新主要 UI 標題、按鈕、參數欄位、狀態 tile 與常用狀態前綴。
- TX/RX 通訊紀錄、M0-M10、版本號、作者與 V/A/W/Ah/Wh 單位不翻譯。
- EXE 外部圖示維持新版去背圖片圖示。

v24 更新：
- EXE 外部圖示改為新版去背圖片圖示，編譯時已嵌入 win32 icon。
- 右欄「輸出控制」標題右側新增「快速輸入」開關。
- 「快速輸入」預設開啟。
- 開啟時，右欄設定電壓、設定電流、LVP、OVP、OCP、OPP 欄位取得焦點/點擊後會自動全選，方便直接覆蓋輸入。
- 快速輸入只全選、不清空，因此單純點擊欄位不會讓欄位進入 dirty 狀態。

v23 更新：
- 「目前調用」從右欄底部移到中間下方狀態區。
- 「目前調用」位於「輸出狀態 / 保護狀態」右側，綠底白字並跨兩格高度。
- 「目前調用」字體加大加粗，作為主要狀態顯示。
- 右欄底部現在只保留「刷新 M0-M10」按鈕。

v22 更新：
- 右欄最下方「目前調用」上方新增「刷新 M0-M10」按鈕。
- 點擊後會暫停自動輪詢，依序讀取 M0-M10 的 V-SET / I-SET。
- 刷新完成後更新快捷組按鈕內的綠色電壓與黃色電流。
- 如果刷新前自動輪詢正在執行，刷新完成後會恢復原本輪詢狀態。

v21 更新：
- 左下角圖片改為新版去背圖片。
- EXE 內嵌主程式圖片同步改為新版，不依賴外部圖片檔。

v20 更新：
- 主圖表 Y 軸改以設定值為基準。
- 左軸電壓預設 0 到目前設定電壓，例如設定 21V 時左軸最高為 21V。
- 右軸電流預設 0 到目前設定電流，例如設定 2A 時右軸最高為 2A。
- 若實際輸出超過設定值，對應 Y 軸會暫時自動放大，避免曲線被頂住或切掉。
- 輪詢讀到設定值、或右欄寫入電壓/電流確認成功後，都會同步更新圖表基準。

v19 更新：
- 主圖表時間軸改成動態視窗，不再一啟動就固定 -60m。
- 剛開始會依資料長度自動使用 10s / 30s / 60s / 2m / 5m / 10m / 15m / 30m / 60m 視窗。
- 跑滿 1 小時後才固定顯示最近 60 分鐘。
- 原始資料與壓縮邏輯維持 v18：原始點超過 1 小時後才壓縮進 3 天分鐘統計。

v18 更新：
- 圖表壓縮邏輯修正為「原始點保留 1 小時，超過 1 小時後才壓縮」。
- 新進資料先只保留在原始區，不再一進資料就同步壓縮。
- 超過 1 小時被移出的 V / A / W 原始點，才合併到分鐘統計資料。
- 右欄「寫入電壓」「寫入電流」只有在寫入後讀回確認一致時，才同步更新下方目前 M 組按鈕的電壓/電流顯示。

v17 更新：
- 主歷史圖表改為最近 1 小時時間窗，X 軸顯示 -60m / -45m / -30m / -15m / now。
- 原始 VOUT / IOUT / POWER 點只保留最近 1 小時，避免長時間輪詢後直接用全部原始點畫圖。
- 背景新增每分鐘壓縮資料：V / A / W 會保存平均、最小、最大，保留最近 3 天。
- 這版先維持主圖表只畫 VOUT 與 IOUT，3 天壓縮資料先作為後續長時間趨勢/統計使用。

v16 更新：

- 按鈕左側顯示 M0-M10，右上顯示綠色電壓，右下顯示黃色電流。
- 快捷組摘要掃描改讀 V-SET 與 I-SET 兩個寄存器，顯示每組電壓與電流。
- 調用某個 M 組讀回成功時，同步更新該按鈕的電壓/電流。
- 右下固定狀態區移除「保護寫入」提示，只保留目前調用。

v15 更新：

- 快捷組摘要掃描中狀態列會顯示「讀取快捷組摘要 M0-M10」。

v14 更新：

- 黑色頂欄改用固定表格排版，修正最大化啟動時「已連線 COMx 115200 8N1」吃字。

v12 更新：

v11 更新：

v10 更新：
- 重新定義快捷組主按鈕與編輯鍵行為。
- 按 M0-M10 主按鈕時：
  1. 先讀取該 M 組參數。
  2. 將 V-SET、I-SET、LVP、OVP、OCP、OPP 反饋到右欄欄位。
  3. 再送出 001DH = M# 調用到從機。
  4. 狀態顯示目前調用：M#。
- 按 M0編-M10編時：
  1. 只讀取該 M 組參數。
  2. 將參數反饋到右欄欄位。
  3. 不送出 001DH，不調用從機。
  4. 狀態顯示正在編輯：M#。

v5 更新：
- 右欄設定電壓、設定電流、OVP、OCP 改成大按鍵步進調整。
- 加減按鈕只改介面數值，必須按「寫入」才會送到設備。
- OCP 維持實測倍率：寫入 A * 1000，讀回 raw / 1000。
- 中央即時參數重新排列，輸出功率改為右側大格顯示。
- 中央新增系統溫度、輸出模式 CV/CC、輸出狀態 ON/OFF。
- 中央新增輸出容量 AH、輸出能量 WH、保護狀態。
- 左欄新增背光亮度、熄屏時間、蜂鳴音、按鍵鎖。
- 左欄新增重啟、恢復出廠設定、校準/清零，按下後會先跳確認視窗。
- 自動輪詢讀取 0000H~0012H，用來同步設定值、輸出值、狀態與累積量。
- 背光、熄屏、蜂鳴音每幾輪同步一次，減少通訊負擔。
- 寫入命令會短暫等待輪詢空檔，降低按下後剛好忙碌而沒送出的機率。

已保留功能：
- COM 埠、波特率、站號選擇。
- Modbus RTU CRC16，低位在前。
- 0x03 讀取、0x06 單暫存器寫入。
- 每次請求最多重試 3 次。
- 底部顯示 TX/RX 原始 hex log。
- 左側統計成功、失敗、CRC 錯誤、重試次數。
- 快捷組 M0-M10 調用後會回讀設定電壓、設定電流、OVP、OCP。

目前寄存器假設：
- 0000H V-SET：raw / 100 V
- 0001H I-SET：raw / 1000 A
- 0002H VOUT：raw / 100 V
- 0003H IOUT：raw / 1000 A
- 0004H POWER：raw / 100 W
- 0006H/0007H AH：32-bit mAh，顯示 Ah
- 0008H/0009H WH：32-bit mWh，顯示 Wh
- 000DH 系統溫度：raw / 10 C
- 000FH 按鍵鎖：0/1
- 0010H 保護狀態
- 0011H CV/CC：0=CV，1=CC
- 0012H 輸出開關：0=OFF，1=ON
- 0014H 背光亮度：0-5
- 0015H 熄屏時間：分鐘
- 001CH 蜂鳴音：0/1
- 001DH 快捷組：M0-M10
- 0020H 恢復出廠設定
- 0021H 校準/清零
- 002FH 重啟
- 0053H + M*0010H OVP：raw / 100 V
- 0054H + M*0010H OCP：raw / 1000 A

注意：
這版仍是原型，建議先用較低電壓電流、小負載測試。若真機顯示與介面倍率不同，再依實測修正。
